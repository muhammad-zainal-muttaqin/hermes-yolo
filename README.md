## 🔄 Reproducing Any Experiment

Each experiment is fully reproducible:

```bash
# Find experiment config
cat experiments/runs/AR_001/config.yaml

# Re-run with same seed
python -c "
from autoresearch import ExperimentRunner, ExperimentConfig
import yaml

# Load config
with open('experiments/runs/AR_001/config.yaml') as f:
    config_dict = yaml.safe_load(f)
    config = ExperimentConfig(**config_dict)

# Run
runner = ExperimentRunner()
results = runner.run_experiment(config)
print(f'mAP50: {results[\"map50\"]}')
"
```
